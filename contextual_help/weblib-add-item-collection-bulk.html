Add items to the collection database in bulk (from a CSV file)

This page allows for uploading a CSV (Comma Separated Value) file containing a number of items to be inserted all at once. The columns supported are:

  1. barcode This is the item's barcode.
  2. title This is the item's title. It is required.
  3. author This is the item's author. It is required.
  4. subject This is the item's subject. It is required.
  5. description This is the item's description.
  6. category This is the item's category.
  7. media This is the item's media.
  8. publisher This is the item's publisher.
  9. publocation This is the item's publisher's location.
  10. pubdate This is the item's publish date.
  11. edition This is the item's edition.
  12. isbn This is the item's ISBN.
  13. type This is the item's type. It is required.
  14. thumburl This is the item's thumbnail URL.
  15. keywords This is the item's keywords (as a quoted, comma separated list).
  16. callnumber This is the item's Call Number.
