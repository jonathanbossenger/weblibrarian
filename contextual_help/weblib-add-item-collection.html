Adding, Viewing, and Editing items in the collection database

This page allows you to view, edit, or add items to the collection database. In edit or add mode, it is also possible to look items up in Amazon's extensive product database and information in Amazon's product database can be used to populate elements of an items record.

The fields of an item's record consist of:

  1. Barcode This is the item's unique identifier, usually a scannable barcode. This field will be automatically generated if left blank. An existing item's barcode cannot be changed.
  2. Title This is the item's title. It is a required field.
  3. Author This is the item's author. It is a required field.
  4. Subject This is the item's subject. It is a required field.
  5. Description This is the item's general description. It is an optional field.
  6. Category This is the item's optional category.
  7. Media This is the item's optional media.
  8. Publisher This is the item's publisher.
  9. Publisher Location This is the item's publisher location.
  10. Publish Date This is the item's publish date.
  11. Edition This is the item's optional edition.
  12. ISBN This is the item's optional isbn.
  13. Type This is the item's required circulation type.
  14. Thumbnail URL This is the item's optional thumbnail image URL.
  15. Keywords These are the item's optional keywords.

On the Add and Edit pages, there is a search box to search Amazon's product database. Searches by title, author, artist, and keywords are possible. Up to 10 results per page are displayed. Individual result items can be looked up in detail or inserted into the item entry form.
