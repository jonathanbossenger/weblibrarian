Aggiungi elementi in blocco al database della collezione (a partire da un file CSV)

In questa pagina si può caricare un file CSV (Comma Separated Value) contenente gli elementi da inserire. Sono supportate le seguenti colonne:

  1. barcode È il codice a barre dell'elemento.
  2. title È il titolo dell'elemento. OBBLIGATORIO.
  3. author È l'autore dell'elemento. OBBLIGATORIO.
  4. subject È l'argomento dell'elemento. OBBLIGATORIO.
  5. description È la descrizione dell'elemento.
  6. category È la categoria dell'elemento.
  7. media È il formato dell'elemento.
  8. publisher È l'editore dell'elemento.
  9. publocation È il luogo di pubblicazione dell'elemento.
  10. pubdate È la data di pubblicazione dell'elemento.
  11. edition È l'edizione dell'elemento.
  12. isbn È il codice ISBN dell'elemento.
  13. type È il tipo di prestito assegnato all'elemento. OBBLIGATORIO.
  14. thumburl Èl'indirizzo dell'immagine collegata all'elemento.
  15. keywords Sono le parole chiave collegate all'elemento (dev'essere un elenco separato da virgole e incluso tra virgolette).
