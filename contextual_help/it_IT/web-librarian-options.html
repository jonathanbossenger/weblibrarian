Schermata delle opzioni del plugin

Sono diponibili 5 opzioni, tutte connesse all'utilizzo dei web service offerti da Amazon. Usando tali servizi si possono ottenere informazioni sugli elementi della collezione da usare nelle pagine di aggiunta e modifica. Le opzioni sono:

  1. Chiave pubblica AWS: È la chiave pubblica che usi per accedere ai web service di Amazon.
  2. Chiave privata AWS: È la tua chiave privata per i web service di Amazon.
  3. Regione AWS: È la regione da usare per i web service di Amazon.
  4. Amazon Associate Tag: È la Amazon Associate Tag da usare.
  5. Debug del database: Permette di accendere o spegnere il debug del database. L'impostazione predefinita è "spento".

Web Librarian funzionerà benissimo anche senza impostare queste opzioni. Semplicemente non si avranno a disposizione gli strumenti di ricerca di Amazon.
