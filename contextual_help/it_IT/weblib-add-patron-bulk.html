Aggiungi iscritti in blocco (a partire da un file CSV)

In questa pagina si può caricare un file CSV (Comma Separated Value) contenente gli elementi da inserire. Il file deve contenere, per lo meno, colonne per nome, cognome, prima riga dell'indirizzo, città, provincia, CAP e numero telefonico. Può anche contenere ID iscritto, altro nome, multe da pagare e scadenza. Se il file non include nella prima l'header delle colonne, il sistema di upload cercherà di indovinare le colonne da solo. Le colonne riconosciute sono:

  • id ID iscritto.
  • firstname Nome.
  • lastname Cognome.
  • extraname Altro nome (per esempio il secondo nome).
  • address1 Indirizzo 1 (la prima riga dell'indirizzo).
  • address2 Indirizzo 2 (la seconda riga dell'indirizzo).
  • city Città.
  • state Provincia (o stato, nel caso di federazioni).
  • zip CAP
  • telephone Numero telefonico.
  • outstandingfines Multe da pagare.
  • expiration Scadenza.
