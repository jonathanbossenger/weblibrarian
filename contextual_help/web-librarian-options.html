The Web Librarian plugin options page

The Web Librarian plugin has five options, all relating to using Amazon's Web Services. These services are used to fetch product information on the collection item add and edit pages. The five options are:

  1. AWS Public Key This is your Amazon Web Services public key.
  2. AWS Private Key This is your Amazon Web Services private key.
  3. AWS Region This is the Amazon Web Services region to use.
  4. Amazon Associate Tag This is the Amazon Associate Tag to use.
  5. Debug Database This is for debugging the database. Normally it is set to off.

The Web Librarian plugin will work just fine without setting these options, you just won't be able to get information about books or other items from Amazon.
